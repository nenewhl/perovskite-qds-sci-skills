# Perovskite QD Skills

A public, sanitized skill collection for perovskite quantum-dot and nanocrystal research workflows.

This repository provides a reusable `pqd-skills` router skill for academic data processing, literature organization, scientific writing, and publication-grade figure preparation in perovskite quantum-dot and related nanocrystal studies.

## Contents

```text
skills/pqd-skills/
├── SKILL.md
└── references/
    ├── workflow.md
    ├── data-analysis.md
    ├── literature.md
    ├── manuscript.md
    ├── figures.md
    ├── submission-review.md
    └── domain-standards.md
templates/
├── python/
└── markdown/
```

## Design principles

- Reproducible data handling before interpretation.
- Journal-style figures with restrained, editable, publication-grade formatting.
- Clear separation between raw data, processed data, fitting decisions, and claims.
- No invented results, policies, citations, or performance values.
- Public-safe content with no private project names, collaborators, unpublished sample IDs, or personal research timelines.

## Quick use

Copy the entire `skills/pqd-skills` folder into your Skills directory or reference it from your agent configuration. Do not copy only `SKILL.md`; the reference files are part of the skill behavior.

## Scope

The skill is intentionally general. It can support perovskite/nanocrystal workflows involving optical spectra, device metrics, stability data, imaging-derived measurements, literature mapping, manuscript drafting, reviewer simulation, and figure quality control.
