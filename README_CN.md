# 钙钛矿量子点 Skills

这是一个公开、去个人化的钙钛矿量子点/纳米晶研究工作流 Skills 仓库，用于数据处理、文献整理、学术写作、SCI 风格绘图和投稿前质量检查。

## 设计原则

- 数据处理优先保证可追溯性，不为了图好看而改变证据。
- 图表应服务科学论点，而不是装饰。
- 区分原始数据、清洗数据、拟合结果和科学结论。
- 不虚构实验结果、期刊政策、引用或性能指标。
- 不包含私人项目、合作单位、样品编号、个人时间线或未公开研究内容。

## 使用方式

复制完整的 `skills/pqd-skills` 文件夹，而不是只复制 `SKILL.md`。`references/` 内文件会影响 skill 的具体行为。
